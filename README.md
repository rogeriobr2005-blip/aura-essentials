# AURA | Essentials & Design
Conceito: E-commerce Minimalista de Curadoria de Interiores.
https://rogeriobr2005-blip.github.io/aura-essentials/

A AURA é uma experiência digital focada no movimento slow living e no design minimalista. A página foi desenvolvida para transmitir sofisticação, calma e exclusividade, utilizando uma estética limpa (estilo escandinavo/moderno) onde o produto é o protagonista absoluto.

Destaques da Experiência do Usuário (UX):
Interface Imersiva: O uso de fontes serifadas (Cormorant Garamond) para títulos e sem-serifa (Inter) para leitura cria um contraste clássico e contemporâneo.

Micro-interações de Luxo: Um cursor personalizado que reage a elementos interativos e efeitos de parallax/reveal ao rolar a página garantem uma sensação de fluidez e "toque artesanal".

Navegação Sem Atrito: Sistema de carrinho lateral (Drawer) que permite ao usuário gerenciar compras sem sair da página atual, mantendo o fluxo de navegação contínuo.

Funcionalidades Técnicas:
Catálogo Dinâmico: Renderização de produtos via JavaScript, facilitando a manutenção e atualização do estoque virtual.

Filtros Inteligentes: Sistema de ordenação por preço e nome, além de busca em tempo real.

Checkout Otimizado: Processo de finalização de compra em duas etapas, com validação de dados e resumo de pedido atualizado via localStorage.

Design Responsivo: Adaptabilidade total para dispositivos móveis, garantindo que a elegância visual se mantenha em qualquer tamanho de tela.

Performance: Uso de técnicas como Lazy Loading para imagens e manipulação eficiente do DOM para garantir carregamento rápido.

Stack Tecnológica:
HTML5 & CSS3: Estrutura semântica e estilização avançada com variáveis CSS e animações complexas.

JavaScript (Vanilla): Lógica de negócios pura, sem dependência de frameworks pesados, focando em performance e controle total da experiência.

Armazenamento Local: Persistência de dados do carrinho entre sessões do navegador.
